Namespace sap.ui.Device.support

Contains information about detected capabilities of the used browser or device.

Field Summary
matchmediaFlag indicating whether media queries via JavaScript are supported.
matchmedialistenerFlag indicating whether events on JavaScript media queries are supported.
orientationFlag indicating whether the native orientationchange event is supported.
pointerFlag indicating whether pointer events are supported.
retinaFlag indicating whether the device has a Retina display.
touchFlag indicating whether touch events are supported.
websocketFlag indicating whether WebSockets are supported.
