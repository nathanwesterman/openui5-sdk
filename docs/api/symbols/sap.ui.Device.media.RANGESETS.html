SAP_STANDARD_EXTENDED
A 4 step range set (Phone, Tablet, Desktop, LargeDesktop).

This range set is initialized by default.
It is similar to the range set SAP_STANDARD but with an additional breakpoint for large desktop screens.
Phone is < 600px
Tablet is 600px >= Tablet < 1024px
Desktop is 1024px >= Desktop < 1440px
LargeDesktop is > 1440px
