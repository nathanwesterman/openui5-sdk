new sap.m.WizardProgressNavigator(sId?, mSettings?)
Constructor for a new WizardProgressNavigator.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties
    • stepCount : int (default: 3)
    • varyingStepCount : boolean (default: false)
  • Events
    • stepChanged : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • stepActivated : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
{string}sId? id for the new control, generated automatically if no id is given
{object}mSettings? initial settings for the new control

getStepCount(): int
Getter for property stepCount.

Sets the total number of steps. Minimum number of steps is 3. Maximum number of steps is 8.

Default value is 3.

Returns:
{int} the value of property stepCount
getVaryingStepCount(): boolean
Getter for property varyingStepCount.

If set to true, this indicates that number of steps can vary. A dashed line is displayed after the last concrete step (set by the stepCount property).

Default value is false.
